# Sistema de Ventas y Gestión – Only-Way Sportswear

## 📌 Descripción

Este proyecto es una aplicación de escritorio desarrollada para optimizar el proceso de ventas y gestión de pedidos de Only-Way Sportswear.

La empresa anteriormente utilizaba procesos manuales como el registro de pedidos en Excel y la facturación escrita a mano, lo que generaba retrasos, redundancia de datos y una comunicación ineficiente.

Para solucionar estos problemas, se desarrolló un sistema centralizado utilizando JavaFX para la interfaz gráfica y MySQL para la gestión de datos.

---

## 🚀 Funcionalidades principales

* Gestión de clientes
* Creación y seguimiento de pedidos
* Sistema de facturación
* Impresión automática de facturas
* Integración con APIs externas
* Notificaciones automáticas sobre el estado de pedidos
* Validación y control de datos
* Generación de reportes y facturas mediante Jaspersoft
 

---

## 🛠️ Tecnologías utilizadas

* **Java** – Lógica principal de la aplicación
* **JavaFX** – Interfaz gráfica de escritorio
* **MySQL** – Base de datos
* **APIs externas** – Facturación digital y notificaciones
* **Jaspersoft** – Generación de reportes y facturación

---

## 🧠 Arquitectura del sistema

El sistema sigue una arquitectura cliente-servidor:

* **Frontend (JavaFX):** Interfaz de usuario
* **Backend (Java):** Lógica de negocio
* **Base de datos (MySQL):** Almacenamiento de clientes, pedidos y facturación

## ⚙️ Flujo del sistema

1. El usuario inicia sesión
2. Se registran clientes y pedidos
3. Los pedidos se almacenan en la base de datos
4. Se generan facturas mediante APIs externas
5. Se envían notificaciones automáticas a los clientes
